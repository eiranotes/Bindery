# 이름 아이디어

`Bindery`는 현재 임시명이다. 아래 후보는 기능 방향과 톤을 기준으로 정리한 것이며, 상표/도메인/앱스토어 중복 검증은 별도 필요하다.

## 방향 1. 로컬 집필 IDE 느낌

| 이름 | 느낌 | 메모 |
|---|---|---|
| **Scriptorium** | 고전적 집필실 | 라틴어계, 장문 창작툴에 어울림 |
| **Manuscript Lab** | 실험실/작업실 | AI 하네스와 분석 기능 강조 |
| **DraftForge** | 초안 단련/제작 | candidate→diff→apply 흐름과 맞음 |
| **StoryForge** | 이야기 제작소 | 직관적이나 유사명 많을 수 있음 |
| **FolioForge** | 원고 묶음/제작 | Scrivener 계열 느낌 |
| **WriteFoundry** | 집필 주조소 | 장기 프로젝트 제작감 |

## 방향 2. 연재/회차 운영 특화

| 이름 | 느낌 | 메모 |
|---|---|---|
| **SerialForge** | 연재 제작소 | 장기 연재 하네스 정체성 강함 |
| **ArcPilot** | 아크 조종/관리 | 플롯·회차 운영 중심 |
| **Episode Works** | 회차 작업장 | 설명적이고 제품 기능이 바로 보임 |
| **ChapterDeck** | 챕터 카드/덱 | Plot Grid, scene card와 어울림 |
| **ArcBinder** | 아크+바인더 | Scrivener식 구조화 강조 |
| **Threadkeeper** | 떡밥/스레드 관리 | open thread 관리 기능과 잘 맞음 |

## 방향 3. Markdown/Obsidian/Git 친화

| 이름 | 느낌 | 메모 |
|---|---|---|
| **InkVault** | 잉크 저장고 | 로컬-first, archive 느낌 |
| **PageVault** | 페이지 보관소 | snapshot/evidence 강조 |
| **StoryVault** | 이야기 저장고 | 직관적이나 유사명 가능성 |
| **InkLedger** | 잉크 장부 | commit journal, canon ledger와 잘 맞음 |
| **CanonLedger** | 설정 장부 | 설정 안정성 특화 느낌 |
| **FictionLedger** | 창작 운영 장부 | evidence-first와 맞음 |

## 이름 선택 기준

1. 장기 연재 도구임이 드러나는가.
2. 단순 AI 생성기가 아니라 운영/검수/상태관리 툴임이 드러나는가.
3. Markdown/Git/local-first 성격과 충돌하지 않는가.
4. 너무 흔한 이름이라 검색성이 떨어지지 않는가.
5. 한국어 제품명과 영어 제품명 중 어느 시장을 우선할지 결정했는가.

## 현재 임시 권장

개발 중 코드명은 다음 중 하나가 좋다.

```text
SerialForge
InkLedger
DraftForge
```

한국어 출시명을 우선하면:

```text
연재공방
원고공방
```
